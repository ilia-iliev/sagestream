# Project requirements:
This project is an Android app that sends push notifications from a list of "quotes"

## The user can do the following actions:
1. Add/Edit/Remove a quote
2. Import quotes from new-line separated text
3. Receive exactly one push notification of a randomly chosen quote at a time of his choosing

## Requirements for notifications
1. The notifications are sent in a random order.
2. The notifications must be scheduled and repeated daily
3. They are push notifications
4. The user must receive the notification at the appropriate time, even when the app is closed
